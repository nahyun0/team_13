하우 메이크 잇(How make it?) 이란?

원하는 요리의 레시피를 단계별로 알려주는 웹사이트

제작 동기

정보제공을 하는 웹사이트를 만들고 싶었으며, 평소 요리를 할때 레시피를 일일히 찾아보는 경우가 빈번히 있는데 이때 일일히 검색해 찾기보단 이를 하나로 모은 레시피 모음집 같은 사이트가 있었으면 좋겠다고 의견을 모았습니다.
다른 사람의 레시피를 보며 나도 하면 해볼까? 하는 마음이 들어 따라 해보게 하는 것이 이 웹사이트의 목적입니다.

기능 소개

  1. 홈페이지
    • Best recipe에 레시피가 담겨져 있는 음식들의 사진을 넣어 한눈에 어떤 레시피가 있는지 보여줍니다.
      -사진 클릭시 -> 해당 레시피 페이지로 이동합니다.

    • 레시피의 종류에는 한식, 중식, 치즈, 베이킹이 있습니다.
      -조원이 각각 하나씩 담당하여 진행하였습니다.

    • 헤더의 글자를 클릭시 각 페이지로 이동합니다.



  2. 각 페이지의 레시피
    • <list> 속 글자를 클릭하면 해당되는 페이지 내의 글로 이동 됩니다.

    • 레시피 밑 참고 블로그 이동, 참고 영상 이동을 클릭하면 더 자세한 레시피를 확인할 수 있습니다.

추가 적으로 구현해 보고 싶은 기능

1) 만든 요리를 블로그처럼 기록이 가능하는 기능
- 로그인 기능을 추가하여 마이페이지에 들어가면 내가 만든 요리를 기록할 수 있으며 자신의 레시피를 올릴 수 있습니다.

2) 다른 사람의 레시피 글을 보며 하트를 누르는 등의 기능을 추가해 레시피를 저장하는 기능

3) 레벨업 기능
-처음 시작할때는 초보 레벨로 시작했다 올리는 요리 글의 수에 따라 세프 등급까지도 갈 수 있도록 그 사이의 등급을 나눠 레벨을 뜨게 하는 것
